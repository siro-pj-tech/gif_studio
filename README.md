# GIFstudio V1.1

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.1-green.svg)

**GIFstudio V1.1** は、ブラウザ上で手軽にGIFアニメーションの生成・編集ができるツールです。シンプルな操作で、誰でも簡単にアニメーション制作を楽しむことができます。

## 🚀 主な機能

- **GIF生成**: 画像シーケンスをアップロードして、素早くGIFを作成。
- **フレーム編集**: 各フレーム間の表示時間（ディレイ）を0.05秒間隔で調整可能。（最大1秒）
- **プレビュー機能**: 保存前にアニメーションの動きをリアルタイムで確認。
- **ダウンロード**: 最適化されたGIFファイルとして書き出し。
- **直感的なUI**: ブラウザベースでインストール不要。

## 🛠 使い方

1. `index.html` をブラウザで開きます。
2. アニメーションに使用したい画像を選択またはドラッグ＆ドロップします。
3. サイズは最初に読み込んだ画像サイズを読み込みますが、指定して拡大・縮小させることも可能です。（比率固定ボタンを実装済み）
4. 設定項目：切り替え速度、最後のフレームから最初に戻るまでの時間を調整します。（時間を延長する時は、最後のフレームが連続して追加されるので容量が増えます）
5. 「GIFを生成」ボタンを押すと、プレビュー画面に移行。完成したファイルを確認し、「保存する」を選択します。
7. ファイル名は自動で作られ、保存確認時に表示されます。「はい」でGIFファイルがダウンロードフォルダに保存されます。
